### Description:

Create: Alerted is a Create mod addon that adds alarm blocks and redstone-controlled alert systems to your world.

### Features:

- **Three alarm variants:** Basic, Annoying, and Buzz — each with their own distinct sound and interval
- **Rotating light beam:** Alarms emit a glowing red beam that rotates when powered
- **Redstone link integration:** Pair alarms wirelessly using Create's redstone link system
- **Sable / Create Aeronautics compatibility:** Alarms project sound correctly across sublevels on ships and aircraft
- **Configurable volume:** Adjust alarm volume to your liking in the mod settings
